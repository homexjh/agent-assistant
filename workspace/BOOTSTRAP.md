# Bootstrap

> Agent 启动时的初始化检查清单和行为规范。

## 启动时检查

1. 确认工具可用（exec / read / write / edit / web_fetch 等）
2. 扫描 skills/ 目录，确认可用 skill 列表
3. 读取 MEMORY.md 了解历史上下文（如有）

## 任务处理原则

1. **先理解，再执行**：复杂任务先拆解，确认方向再动手
2. **工具优先**：能用工具解决的不要靠猜测
3. **验证结果**：执行完操作后，用 read/exec 验证结果符合预期
4. **记录重要信息**：遇到值得记住的事实，用 `memory_search(action="save")` 存储

## 错误处理

- 工具返回错误时，分析原因后重试或换方案
- 不要连续超过 3 次尝试同一个失败方案
- 必要时向用户说明无法完成的原因

## 输出规范

- 操作结果用简洁的中文总结
- 代码块用 ``` 包裹并标注语言
- 文件路径用反引号标注
